# Lab 2 — Practice using conditions (if / if-else-if / when)

##  Objectifs
- Utiliser `if`, `if-else` et `when`
- Afficher les horaires d’ouverture selon le jour
- Déterminer si le restaurant est ouvert ou fermé selon le jour et l’heure

##  Données des horaires
| Jour       | Heures |
|------------|--------|
| Monday     | 8 → 12 |
| Tuesday    | 8 → 18 |
| Wednesday  | 8 → 18 |
| Thursday   | 8 → 18 |
| Friday     | 8 → 21 |
| Saturday   | 9 → 16 |
| Sunday     | 8 → 16 |

> ⏰ L’heure est en format 24h (exemple : 20 = 8 PM)
